# 📊 Superstore Sales Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Superstore Sales dataset to extract meaningful business insights using Python.

The goal is to understand sales performance, profit trends, customer segments, and detect patterns or anomalies in the data.

---

## 🎯 Objectives

* Understand dataset structure and variables
* Analyze sales and profit across categories
* Identify top-performing regions and products
* Detect trends over time (monthly sales)
* Find loss-making products and outliers
* Study relationship between sales and profit

---

## 🛠 Tools & Libraries Used

* Python
* Pandas
* Matplotlib

---

## 📂 Dataset Information

The dataset contains information about:

* Orders and shipping details
* Customer segments
* Product categories
* Sales, profit, discount
* Regional data

---

## 🔍 Analysis Performed

### 1. Data Cleaning

* Checked missing values
* Checked duplicate records
* Converted date columns

### 2. Exploratory Analysis

* Sales by Category
* Profit by Category
* Sales by Region
* Sales by State
* Monthly Sales Trend

### 3. Business Insights

* Top performing categories
* Loss-making products
* High sales regions
* Sales vs Profit relationship

### 4. Visualization

* Bar charts for categories and regions
* Line chart for sales trends
* Scatter plot for correlation
* Boxplot for outliers

---

## 📊 Key Insights

* Certain categories generate high sales but low profit
* Some products consistently show losses
* Sales vary significantly across regions
* Strong relationship observed between sales and profit
* Outliers exist in sales distribution
